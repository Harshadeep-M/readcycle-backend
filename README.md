# ReadCycle Backend

Spring Boot backend for the ReadCycle recycling platform.

## Tech Stack
- Java
- Spring Boot
- MySQL
- JPA / Hibernate
- REST APIs
- Maven

## Features
- Create user
- Get all users
- Get user by ID
- Update user
- Delete user

## API Endpoints

GET /users  
POST /users  
GET /users/{id}  
PUT /users/{id}  
DELETE /users/{id}
